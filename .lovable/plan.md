

# Campus Connect — Modernized

A safe, anonymous campus chat app for **@srmap.edu.in** students with peer matching, friends, and group chats. Built with a modernized dark purple aesthetic.

---

## 1. Authentication (Firebase)
- Email/password login and signup using your existing Firebase project
- Restrict registration to **@srmap.edu.in** emails only
- Logout functionality
- Clean, modern login/signup screens with the purple gradient theme

## 2. Anonymous Peer Matching & Chat
- **Connect button** to enter a matchmaking queue
- Real-time matching with another waiting student
- **Anonymous 1-on-1 chat** with:
  - Real-time messaging via Firebase Firestore
  - Typing indicators (animated dots)
  - Bad word filter with 30-second mute
  - "End Chat" button that notifies the peer
- Smooth loading/matching animation while waiting

## 3. Friends System
- **Send friend request** to your current anonymous chat peer
- **Friend Requests** tab showing incoming requests (accept/decline)
- **Friends list** showing accepted friends
- **Private chat** with friends (non-anonymous, persistent)

## 4. Group Chats
- **Create Group** with a name
- Invite friends to groups
- Group chat with real-time messaging
- View list of your groups

## 5. Navigation & Layout
- **Bottom navigation bar** with tabs: Chat, Friends, Groups
- Mobile-first responsive design
- Screens transition smoothly between views

## 6. Modernized Purple Theme
- Dark background with refined purple (#8c52ff) accents
- Glassmorphism-inspired cards and containers
- Polished message bubbles with subtle animations
- Clean typography and spacing
- Custom scrollbars matching the theme
